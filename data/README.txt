1.Thư mục boluat chứa các file docx về các bộ luật của Việt Nam (Đã được xử lý). 
  Dữ liệu này dùng để huấn luyện không giám sát để đạt được model ViLawT5-base.
  Link: https://www.kaggle.com/datasets/thuntrngbnh/data-luat-qa

2.Thư mục xml_data chứa các file xml về các dữ liệu hỏi đáp trong Bộ Luật Dân Sự 2015. 
  Trong đó các tên của các file xml được đặt theo số chương trong Bộ Luật Dân Sự 2015. 
  Dữ liệu này dùng để fine tuning để đạt được các model ViT5-QAChatbot, ViLawT5-QAChatbot, ViLawT5-RL.
  Link: https://www.kaggle.com/datasets/thuntrngbnh/xml-data-luat-qa

3.Thư mục reward_data chứa file reward_res_sheet. 
  Đây là dữ liệu đánh giá phản hồi của người dùng dùng để huấn luyện Reward Model.
  Link: https://www.kaggle.com/datasets/thuntrngbnh/reward-data
